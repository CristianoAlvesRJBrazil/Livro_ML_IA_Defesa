# Inteligência Artificial e Machine Learning Avançado para Defesa

Notebooks didáticos do curso **Inteligência Artificial e Machine Learning Avançado para Defesa** — Prof. Cristiano Alves · Quantum Strategic AI.

Cada capítulo do livro é acompanhado de uma aula prática em formato Google Colab, com teoria, código executável em `scikit-learn`, visualizações e exercícios (Essencial · Tático · Estratégico).

## Como usar

1. Abra o notebook desejado no [Google Colab](https://colab.research.google.com/) (menu *File → Open notebook → GitHub* e cole a URL deste repositório), ou clique no arquivo aqui no GitHub e use o botão *Open in Colab*, se disponível.
2. Execute as células em ordem (`Runtime → Run all`). Não é necessária nenhuma instalação: os notebooks usam apenas `numpy`, `matplotlib` e `scikit-learn`, já presentes no Colab.
3. As células dos exercícios trazem marcações `# <--- ALTERE AQUI` indicando o que modificar.

## Reprodutibilidade

Todos os notebooks fixam a semente aleatória (`SEMENTE = 0`) — dados os mesmos dados, os mesmos resultados. Em sistemas de defesa, reprodutibilidade não é opcional.
